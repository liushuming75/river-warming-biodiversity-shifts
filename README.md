# river-warming-biodiversity-shifts
This repository contains code for analyzing long-term changes in European freshwater fish biodiversity in response to river warming.

## Structure

- `src/01_trend_analysis/`: Trend estimation using LMMs
- `src/02_phase_analysis/`: Phase-specific modelling
- `src/03_functional_and_uniqueness/`: Functional diversity and LCBD
- `src/04_main_figures/`: Main figures
- `src/05_extended_figures/`: Supplementary figures

## Data

Due to data size limitations, only the data required to run the code are provided in this repository.

The full datasets supporting the findings of this study are publicly available at Zenodo:
https://doi.org/10.5281/zenodo.19159931
